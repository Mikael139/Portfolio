Hard skills

Angular

Angular é uma plataforma e framework para construção da interface de aplicações usando HTML, CSS e, principalmente, JavaScript baseado em TypeScript, criada pelos desenvolvedores da Google

JAVA

Java é uma linguagem de programação orientada a objetos desenvolvida na década de 90 por uma equipe de programadores chefiada por James Gosling, na empresa Sun Microsystems, que em 2008 foi adquirido pela empresa Oracle Corporation.

SQLServer

O SQL Server é um sistema gerenciador de bancos de dados relacionais (SGBDR). Com ele podemos criar bases de dados com tabelas e manipular seus respectivos dados. Dentre as operações que podemos realizar com as tabelas podemos citar uma bastante conhecida como CRUD (CREATE, READ, UPDATE, DELETE).

GIT

O Git é uma ferramenta que ajuda a gerenciar o código-fonte e seu histórico de mudanças. Ele permite que vários desenvolvedores trabalhem no mesmo projeto ao mesmo tempo sem se atrapalharem. Com o Git, é possível criar diferentes versões do código (chamadas de branches) e depois juntá-las novamente (o que é chamado de merge). Isso torna o desenvolvimento de software mais eficiente e organizado.
